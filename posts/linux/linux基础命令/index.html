Linux基础命令
linux基础命令,基于manjaro发行版学习
2021-10-08 17:24 folder linux   label linux   schedule 37 s. 
# 查看内核信息
uname -a
# 查看端口号
-t (tcp) 仅显示tcp相关选项
-u (udp)仅显示udp相关选项
-n 拒绝显示别名，能显示数字的全部转化为数字
-l 仅列出在Listen(监听)的服务状态
-p 显示建立相关链接的程序名
netstat -tunlp | grep 端口号
# 查看当前所有tcp端口
netstat -ntlp
# 查看所有80端口使用情况
netstat -ntulp | grep 80
